Admin Notes

The dashboard is password-protected and backed by SQLite. Notification delivery status is saved with each quote request so email issues are easier to spot and fix.

  • Set `ADMIN_PASSWORD` to change the starter password.
  • Use `RESEND_API_KEY` and `RESEND_FROM` on Render for reliable hosted email alerts.
  • Quote requests are stored in `data/rampal-quote-requests.db`.
